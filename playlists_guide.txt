Playlist Settings:
- Public visibility
- Add description for each playlist
- Add relevant keywords
- Set playlist thumbnail (use video thumbnails)
- Enable "Allow embedding" for wider reach

Playlist Descriptions Example:
"Cuentos de Baba Yaga: Las historias más aterradoras de la bruja del bosque eslavo. Sumérgete en el mundo de Baba Yaga y su cabaña de patas de pollo."

Thumbnail Strategy:
- Use the most scary/engaging thumbnail from videos in playlist
- Consistent style across all playlists
- Include playlist name in thumbnail text
